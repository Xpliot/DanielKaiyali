Lod library

The library is divided into four areas: Foundation, Geometry, Graphing and Statistics. Foundation obtains basic operations; for instance: adding two numbers/an entire array, Multiplying two numbers/array, calculating power, square-root, Pythagorean theorem, sine, cosine, tangent, etc. Geometry contains Formulas for properties of 2D and 3D shapes, for example, volume, area, surface area, perimeter, etc. Graphing displays three types of graphs: bar graph, line graph, and pie graph. The Statistics is a mandatory persona of Lod library. This bit will find/solve numerous equations, for instance, finding the mode, range, mean, the max/min number, solving patterns, random number picker and many more methods. These classes make things easier for users to use more than one array type, such as int arrays, double arrays, and long arrays to manipulate data.

Visua2D

Is an application developed in the java programming language is the first-ever collaboration between Daniel Kaiyali and Archisman Nath with the intention of establishing a graphical, mathematical Engine that enables the audience to solve their math problems especially in the field of 2D geometry. This application also contains several geometric formulas as well as graphic representations of how 2D shapes appear. Users can interact with 2D shapes within the application while exploring the numerous different functionality options.

The software also includes a real-time view of the area, length, width, height, and other features of geometric shapes, which replies in an immediate response based on the user's interactions with the shape. Overall, this application aims to provide a visual presentation of shapes and mathematical functions to be useful for students and academic purposes.
