Lod library

is an advanced math library; it displays graphs inside a swing application, solves properties of two/three-dimensional (2D/3D) shapes, and is a more accessible tool in various Array types as well as a tool for data manipulation.

The library is divided into four areas: Foundation, Geometry, Graphing and Statistics. Foundation obtains basic operations; for instance: adding two numbers/an entire array, Multiplying two numbers/array, calculating power, square-root, Pythagorean theorem, sine, cosine, tangent, etc. Geometry contains Formulas for properties of 2D and 3D shapes, for example, volume, area, surface area perimeter, etc. Graphing displays three types of graphs: bar graph, line graph, and pie graph. The Statistics is a mandatory persona of Lod library. This bit will find/solve numerous equations, for instance, finding the mode, range, mean, the max/min number, solving patterns, random number picker and many more methods. These classes make things easier for users to use more than one array type, such as int arrays, double arrays, and long arrays to manipulate data.

The Visua2D application developed using the Java programming language was the first-ever collaboration between Daniel Kaiyali and Archisman Nath to develop a mathematical application that shows the audience step-by-step how to solve their math problems. This application also has numerous geometric formulas and includes visual exhibits of how 2D shapes look. The application allows users to interact with 2D shapes while exploring the numerous different functionalities options. A live view of the area, length, width, height, or any other properties of geometric shapes will be updated constantly based on the user's interactions with the shape.
